Proyecto Uniformatrioska


Proyecto de automatización de procesos, enfocado en la creación de una herramienta para optimizar la gestión y el registro de tiempos de producción. Con esta herramienta, los usuarios pueden registrar automáticamente, con solo presionar un botón, una marca de tiempo que permite calcular de manera eficiente los tiempos de producción en cada etapa del proceso. El desarrollo de la herramienta se realizó utilizando Python para la lógica del backend, junto con HTML, Bootstrap 5 y CSS para la interfaz de usuario, asegurando una experiencia visual moderna y adaptativa. Como base de datos, se emplea una hoja de cálculo de Google Sheets, lo que permite que la información esté siempre disponible y accesible para el cliente en cualquier momento y desde cualquier lugar, facilitando el monitoreo y control en tiempo real.

Tapiceria Resurreccion


Desarrollo de una plataforma web integral para la gestión de entradas, salidas, inventario y ventas. Esta plataforma está diseñada para ofrecer un control eficiente y centralizado de todas las operaciones relacionadas con la gestión de productos y ventas, optimizando tanto el flujo de inventarios como el registro de transacciones. El proyecto fue desarrollado utilizando Python y el framework Django para el backend, mientras que HTML, Bootstrap 5 y CSS se emplearon para crear una interfaz de usuario moderna y responsiva. La base de datos utilizada es MySQL, lo que permite un manejo robusto y escalable de la información, asegurando integridad y rendimiento en la gestión de los datos comerciales y de inventario.
